# BaseFinder Mod

Mod de Minecraft Fabric 1.21.5 para encontrar bases en survival.

## Características
- HUD en pantalla
- Escaneo de 10k+ chunks (cache persistente)
- ESP con cajas y tracers
- Compatible con todos los mods

## Controles
- `-` (menos): Activar/desactivar mod
- `=` (igual): Activar/desactivar tracers
- `]` (corchete derecho): Activar/desactivar HUD
- `[` (corchete izquierdo): Cambiar radio de escaneo

## Compilación automática
Este repositorio usa GitHub Actions para compilar el mod automáticamente.

Ve a la pestaña **Actions** → **Build BaseFinder Mod** → última ejecución → **Artifacts** para descargar el `.jar`.
